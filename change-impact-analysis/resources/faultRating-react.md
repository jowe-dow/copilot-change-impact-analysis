# React 项目功能影响等级表

> **AI 使用说明**：在影响分析阶段，根据变更文件关联的受影响菜单/功能，在此表中查找对应的影响等级（p0/p1/p2/p3）。
> 若受影响的功能**不在此表中**，默认评定为 **p3**。

## 等级说明

| 等级 | 含义 | 回归策略 |
|------|------|---------|
| 🔴 P0 | 入口阻断 / 核心功能阻断 | 立即回归，阻塞发布 |
| 🟠 P1 | 重要功能受影响 | 发布前必须回归 |
| 🟡 P2 | 非重要功能受影响 | 可安排回归，不阻塞发布 |
| ⚪ P3 | 其他 / 不在等级表中 | 低优先级，可观察 |
