# Change Impact Analysis — 示例文档导航

## 📁 文件列表

| 文件名 | 用途 | 适用场景 |
|--------|------|---------|
| `scenario-standard-analysis.md` | 标准分析流程 | PR 提交前的完整影响评估 |
| `scenario-regression-advice.md` | 回归建议生成 | 为 QA 生成测试回归计划 |
| `scenario-route-mapping.md` | 路由映射表生成 | 独立生成/更新路由-菜单映射表 |
| `common-errors.md` | 常见错误 | 分析过程中的坑和解决方案 |
| `reference.md` | 速查表 | 命令、脚本、配置速查 |

## 🎯 快速导航

### 按任务类型

- **我要分析 PR 影响范围** → `scenario-standard-analysis.md`
- **我要给测试提供回归建议** → `scenario-regression-advice.md`
- **我只需要路由映射表** → `scenario-route-mapping.md`

### 按问题类型

- **分析结果不准/遗漏** → `common-errors.md`
- **脚本报错/命令不对** → `reference.md`
- **菜单路径映射不对** → `scenario-route-mapping.md` + `common-errors.md`

## 📖 推荐阅读顺序

### 新手路径
1. `scenario-route-mapping.md` — 先理解路由映射机制
2. `scenario-standard-analysis.md` — 学习完整分析流程
3. `scenario-regression-advice.md` — 学习如何生成回归建议
4. `common-errors.md` — 了解常见陷阱

### 进阶路径（熟悉流程后）
1. `reference.md` — 查命令和配置
2. 按需查看具体场景文档
